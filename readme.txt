Application de click and collect pour les petits commerçants. Inscription et vitrine gratuits sur www.localdrivy.fr. Gérez vos produits, stocks et commandes depuis le backoffice web ou mobile.

== Description ==
- Localdrivy est un connecteur des comptes commerçants inscrits sur www.localdrivy.fr
- Ce plugin permet de publier la boutique et les horaires à partir de blocks Elementor
- Ce connecteur permet d'intégrer aux site wordpress la page boutique, le service de click and collect et le paiement en ligne
- Localdrivy click and collect est adapté aux petits commerçants comme les restaurateurs, les boucheries, les pizzerias, les fleuristes
- L'inscription et la vitrine de produits sont gratuits sur pro.localdrivy.fr/signup
- Les clés apis se trouvent sur le backoffice pro.localdrivy.fr/api-integration
- Les conditions de l'application et du site localdrivy se trouve sur www.localdrivy.fr/assets/docs/cgu-cgv-vendeurs-acheteurs-localdrivy.pdf
- Pour toutes remarques ou problèmes laissez nous un message sur notre formulaire de contact www.localdrivy.fr

[youtube https://www.youtube.com/watch?v=mg3FAv8f1YY]

== Installation ==
Téléchargez  "Localdrivy" et décompressez le dossier in wp-content/plugins
Activez le plugin depuis la gestion des extensions wordpress (/wp-admin/plugins.php)
Entrez la clé api dans les paramètres Localdrivy  (/wp-admin/options-general.php?page=localdrivy+options)
